Пользовательские функции
При обработке данных в прошрамме Excel, иногда, нам приходится выполнять такие операции, для выполнения которых требуются нестандартные операции. Такие операции можно выполнять с использованием всех возможностей самой программы, при этом различные функции, имеющиеся в Excel, используются с определённой комбинацией, либо операции можно выполнять с использованием одной функции, внутри которой используются и другие (стандартные), а сама функция считается пользовательской. Иногда с использованием стандартных функций выполнять такие операции невозможно. 
Нестандартные функции создаются следующим образом: 
- Необходимо открыть редактор языка VBA(Visual Basic for Application), либо сочетанием клавиш <Alt + F11>, либо нажатием на значок <Visual Basic> во вкладке <Разработчик>;
- Необходимо добавить новый модуль и написать в нём код функции;
Добавление модуля можно выполнять разными способоами:
1) Меню Insert -> Module
