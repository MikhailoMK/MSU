Календарь событий
Реализовать на веб-странице календарь, в котором должны иметься следующие функциональности: добавлние события для определённого дня с указанием даты и времени проведения события, напоминание о событии с настройкой интервала, редактирование слбытия и удаление события. Кроме того для определённых событий должен существовать параметр для переноса события на другой год(ы). Для сохранения всеё информации использовать файл с определённой структурой.
