Тема: Назначение, применение, синтаксис и возможности языка JavaScript

Многие современные технологии которые сделали возможным само существование WWW давно превзошли возлагавшиеся на них первоначальные надежды. В начале Web разрабатывалась как сетевая среда управления статическими текстами и изображениями. Она постоянно улучшается, изменяется и популяризуется разработчиками Web-технологий. Для улучшения и достижения поставленных целей разработчики выполняют различную работу и стараются ещё больше расширить использование ресурсов Web. Один из основных современных технологий который дал разработчикам осуществить свои идеи выполнять различные нестандартные действия в пространстве Web является язык JavaScript. После внедрения в компьютер клиента этот язык может превратить страницу со статическим содержимым в привлекательный интерактивный и весьма разумный документ. Приложения - так называемые сценарии JavaScript могут выполнять различные операции как приветствие, приведение в движение различных объектов изменение размеров цвета и других различных параметров объекта которые сделают веб-страницу более наглядной. Наряду с языком JavaScript существуют и другие языки при помощи которых обеспечивается полноценное использование ресурсов веб-страниц. Один из способов повышения эффективности работы по управлению пользователем содержимой страницы является установление с сервером соединения которое обслуживает страницу. Предназначение серверных сценариев - это самое широкое начиная с проверки введённого имени или пароля до задержки выполнения определенного действия написание специальных серверных сценарии обычно требует навыков программирования. Ряд таких сценариев с использование фундаментальных языков программирования таких как PL, Python, Java, C, C++. Разработчику не удается самостоятельно создать серверные приложения если он не знаком с технологиями php, ASP.NET, JSP и callfusion. Многие серверы позволяют выполнять серверные сценарии

Типы данных в языке JS
Для написания кодов и их подключении к веб-странице необходимо использовать соответствующую команду, которая относится к языке html, которую называют тегом и используется в следующем виде <script>...</script>. В качестве типов данных в языке JS могут использоваться следующие типы:
1. Строковой - любая последовательность символов, заключённых в кавычки;
2. Числовой - любое число, указанное без кавычек;
3. null - отсутствие значения (пустота);
4. Объект - некоторый программный объект определяемый своими свойствами и методами. Массивы также являются объектами;
5. Функция.
Если во время написания кода пользователю необходимо задавать число с помощью текстовых полей формы, то JS такие значения принимает как строку и чтобы значения использовалось как число его необходимо перевести к числовому типу. Для того, чтобы объявить переманенные, которые могут использоваться во всех случаях необходимо использовать ключевое слово "var"
