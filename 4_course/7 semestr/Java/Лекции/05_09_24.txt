При объявлении классов используемых компонент необходимо добавить ключевое слово "final" для того, чтобы методы и св-ва компоненты были доступны в других классах, кроме того, если класс, внутри которого используется компоненты, является статическим, то при создании дополнительных классов, а также объявлении компонент необходимо добавить ключевое слово "static".
	При первом способе использовании компонент, который будем называть "визуальным добавлением компоненты", в проекте необходимо добавить новый файл в виде отдельного класса и при необходимости сделать его основным. Далее из списка, путём перетаскивания, добавить на форму необходимые компоненты. Надо отметить, что добавляемый новый класс представляет собой форму, которая считается самым основным компонентом.
	Второй способ использования компоненты - набор соответствующего кода, который является ручным способом использования компонент.
