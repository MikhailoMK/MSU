Операторы языка Java.
	В языке Java операторы как и в других языках программирования имеют определённые имена и делятся на группы. Синтаксис использования операторов является точно таким же как в языке C++.

В группу условных операторов входят такие операторы как:
if ... else ... - условный оператор;
? ... : ... - тернарный оператор;
switch ... case ... default ... - в свою очередь является оператором выбора.

К группе циклических операторов относятся операторы:
for - с параметром;
while - с предусловием;
do - while - с постусловием.

Опероторы перехода:
break
continue
break goto
return

{ } - оператор для указания начала и конца выполнения операций.

Классы.
	В языке Java классы используются аналогично как и в языке C++, за исключением команды extends, которая предназначена для объявления базового класса в классе "наследники".

Визуальный режим программирования. Библиотека SWING.
	В языке Java тоже поддерживается визуальный режим программирования, в нём для использования визуальных компонент именно в среде NetBeans существует 2 модуля, позволяющие использовать компоненты, AWT и SWING. У модуля AWT сокращённый набор компонент, которые связаны с компонентами ОС. Модуль SWING в отличие от AWT имеет широкий набор компоненит с возможностью различных способов их настройки. Компоненты в визуальном режиме в среде NetBeans можно использовать двкмя способами:
1) После создания обычного проекта можно вручную написать код добавления и настройки компоненты;
2) Во время создания проекта можно добавить в проект класс формы с возможностью визульного добавления компоненты из предлагаемого набора.

Как и в других языках, основным объектом является форма, которая в языке Java называется Frame. При использовании компонент библиотеки SWING необходимо перед названием класса добавить "J".
