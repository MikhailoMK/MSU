Java-технологии
Java - ООЯП, разработанный компанией Sun Microsystems, которая в последующем была приобретена компанией Oracle. Официальная дата выпуска языка 23 мая 1993 года (23.05.1991). Изначально язык назывался Oak, разработчиком которого является Джейсон Гослинг. Язык был разработан для программирования бытовых электронных устройств. В последствии он был переименован в Java и стал использоваться для прораммирования клиентских и серверных ПО. Название языка исходит от марки кофе, поэтому официальной эмблемой является чашка с дымящимся кофе. Приложения Java обычно транслируются в специальный байт-код, поэтому они могут работать на любой виртуальной Java-машине, вне зависимости от компьютерной архитектуры. Существует следующая схема, которая описывает структуру и процесс преобразования кодов Java и всех соответствующих механизмов:
1) Платформа X: Текст программы на языке Java -> JDK (Java Developer Kit);
2) Байт-код Java;
3) Платформа Y: JRE (Java RinTime Enviroment), JVM(Java Virtual Machine).

Java внутри себя имеет несколько различных основных семейств технологий:
1) Java SE (Standart Edition) - является основным изданием Java, содержит внутри себя компиляторы API, JRE и подходит для создания пользовательских приложений, в основном рпиложения для десктопных систем;
2) Java EE (Enterprise Edition) - представляет собой набор спецификаций для создания ПО уровня предприятия;
3) Java ME (Micro Edition) - технология создана для использования в устройствах ограниченных по вычислительной мощности;
4) JavaFX -  является технологией, которая считается следующим шагом в эфолюции Java, как Rich Client Platform, и предназначена для создания графических интерфейсов корпаративных приложений
и бизнеса;
5) Java Card - технология представляет безопасную среду для приложений работающих на SIM-картах и других устройствах с очень ограниченным объёмом памяти и возможносятми обработки;
6) Java-Web - используется для создания веб-сайтов и/или веб-приложений;
7) Java и Android - язык активно используется для создания мобильных приложения для ОС Android, при этом программы компилируются в нестандартный байт-код для использования их виртуальной машиной Davlik. Для осуществления такой компиляции используется дополнительный инструмент под названием SDK (Software Development Kit), разработанный компанией Google. В данном случае разработку приложений можно осуществить в средах Andriod Studio, NetBeans, IntelliJIDEA или Eclipse с использованием плагина ADT (Andriod Development Tools). Во всех случаях версия JDK должна быть 5.0 или выше.

Существуют несколько основных возможностей осуществляемые использованием языка Java:
1) Автоматическое управление памятью;
2) Расширенные возможности обработки исключительных ситуаций;
3) Богатый набор средств фильтрации ввода/вывода;
4) Набор стандартных коллекций;
5) Наличие простых средств создания сетевых приложений (в том числе с использованием протокола RMI);
6) Наличие классов, позволяющих выполнять HTTP-запросы и обрабатывать ответы;
7) Встроенные в язык средства создания многопоточных приложений, которые потом были портированы на многие языки;
8) Унифицированный доступ к БД:
	А) На уровне отдельных SQL-запросов, на основе JDBC или SQLI;
	Б) На уровне концепции объектов обладающих способностью к хранению в БД, на освнове Java Data Objects и Java Persistence API;
9) Параллельное выполнение программ.

Существует несколько различных видов и версий ПО, которые используются как среда разработки Java:
1) JDK - помимо набора библиотек для платформ Java SE и Java EE, содержит компилятор командной строки под названием javac (java common) и набор утилит также работающих в режиме командной строки;
2) NetBeans IDE - свободная интегрированная среда разработки для всех платформ Java. Пропагандируется корпорацией Oracle, как базовые средства для разработки ПО на языке Java и других языках;
3) Eclipse IDE - свободная интегрированная среда разработки для Java (SE, EE, ME). Пропагандируется корпорацией IBM, которая является одним из важнейших разработчиков корпаративного ПО, как базовые средства разработки ПО на языке Java и других языках;
4) IntelliJ IDEA - среда разработки для платформ Java (SE, EE, ME). Её разработчиком является компания JetBrains. Распространаяется в двух версиях: свободной/беспалтаной (Community Edition) и коммерческой/проприетарной (Ultimate Edition);
5) JDeveloper - среда разработки для платформ Java (SE, EE, ME). Её разработчиком является компания Oracle;
6) BlueJ - среда разработки ПО на языке Java, созданная в основном для использования в обучении, но также подходит для разработки небольших программ;
6) Java для IOS - обучающее приложение и компилятор для IOS;
7) Geany - свободная среда разработки ПО, написанная с использованием библиотеки GTK2.
